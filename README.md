# Anvesha AI

Anvesha AI is a sovereign Indian search engine powered by open-source technologies, designed with a focus on privacy and local relevance.

## Architecture

- **Frontend:** Next.js (Standalone)
- **Backend:** FastAPI (Gunicorn + Uvicorn workers)
- **Search Engine:** SearxNG (Optimized for India)
- **Deployment:** Docker Compose on Hugging Face Spaces
